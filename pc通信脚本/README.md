## android pc通信脚本
### 功能
>前提条件:mac linux adb

> 功能: 无需安装微信，登录等麻烦操作，只需执行脚本实现pc与andorid手机的字符串传输。
### 实现思路

通过adb 与andorid广播机制来达到相互通信的目的
